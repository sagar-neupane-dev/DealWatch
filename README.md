## 🚀 Tech Stack

- **Frontend**: Next.js + TypeScript + React + Tailwind CSS
- **Backend**: Serverless (Vercel Functions)
- **Database**: MongoDB for price history storage
- **Web Scraping**: Axios + Cheerio
- **AI Integration**: Groq AI API for automated product descriptions
- **Deployment**: Vercel with CI/CD and cron jobs

---

## 💎 Key Features

- **📊 Price Drop Tracking** - Real-time price monitoring and drop detection
- **🔔 Notification System** - Instant alerts when prices meet your targets
- **📈 Historical Price Visualization** - Visual trends and analytics
- **🤖 AI-Powered Summaries** - Automated product insights using Groq
- **⚡ Optimized Scraping** - Lightweight data collection with Axios + Cheerio

---

## 🎯 Technical Highlights

**Challenge**: Puppeteer was too heavy for Vercel's serverless runtime.

**Solution**: Migrated to Axios + Cheerio, eliminating memory constraints and cold-start issues entirely.
